# Project Charter

**Projeto:** Fraud Analytics Platform

**Subtítulo:** End-to-End Data Platform for Fraud Detection and Business Intelligence

**Versão:** 0.1
**Status:** Em desenvolvimento
**Data de criação:** Julho de 2026

---

# 1. Visão Geral

A Fraud Analytics Platform é uma plataforma analítica desenvolvida para apoiar a identificação de fraudes em transações financeiras, simulando um ambiente corporativo de uma instituição financeira digital.

O projeto contempla todas as etapas do ciclo de vida de dados, desde a ingestão e preparação das informações até a disponibilização de modelos preditivos e dashboards para suporte à tomada de decisão.

Além do desenvolvimento técnico, a iniciativa tem como objetivo demonstrar boas práticas de Engenharia de Dados, Data Science, Business Intelligence e documentação de projetos.

---

# 2. Empresa

Este projeto foi concebido para atender às necessidades da **FinTrust Bank**, uma fintech fictícia especializada em serviços financeiros digitais.

A empresa oferece produtos como conta digital, cartões de crédito, transferências instantâneas e empréstimos pessoais. Com o crescimento do número de clientes e do volume de transações, a organização passou a enfrentar um aumento significativo nas tentativas de fraude.

Como consequência, tornou-se necessário desenvolver uma plataforma analítica capaz de apoiar a identificação de comportamentos suspeitos e fornecer informações estratégicas para as equipes de risco e gestão.

---

# 3. Objetivo Geral

Projetar e implementar uma plataforma de Analytics capaz de integrar dados, automatizar processos de ETL, aplicar modelos de Machine Learning para detecção de fraudes e disponibilizar informações estratégicas por meio de dashboards interativos.

---

# 4. Objetivos Específicos

* Construir um pipeline completo de ingestão e transformação de dados.
* Garantir a qualidade e a organização das informações.
* Estruturar um ambiente analítico utilizando SQL Server.
* Desenvolver um Data Warehouse em modelo dimensional.
* Treinar modelos de Machine Learning para classificação de fraudes.
* Disponibilizar dashboards executivos em Power BI.
* Integrar IA Generativa para auxiliar na interpretação dos resultados.
* Documentar todas as etapas do projeto.

---

# 5. Escopo

O projeto contempla:

* Ingestão dos dados do dataset IEEE-CIS Fraud Detection.
* Processos de ETL utilizando Python.
* Automatização de cargas utilizando SSIS.
* Armazenamento em SQL Server.
* Construção do Data Warehouse.
* Desenvolvimento de modelos preditivos.
* Construção de dashboards em Power BI.
* Documentação técnica completa.

---

# 6. Fora do Escopo

Não fazem parte deste projeto:

* Integração com sistemas bancários reais.
* Processamento de dados em tempo real.
* Desenvolvimento de aplicações web.
* Autenticação de usuários.
* Implantação em ambiente de produção.

---

# 7. Tecnologias

* Python
* SQL Server Express
* SQL Server Integration Services (SSIS)
* SQL (T-SQL)
* Scikit-learn
* Power BI Desktop
* Git e GitHub
* BigQuery Sandbox
* Ollama (IA Generativa)
* Visual Studio Code
* Jupyter Notebook

---

# 8. Resultado Esperado

Ao final do projeto será disponibilizada uma plataforma demonstrativa capaz de reproduzir um fluxo corporativo de Engenharia de Dados, Analytics e Data Science, permitindo identificar padrões de fraude, apoiar a tomada de decisão e servir como referência técnica para estudos e portfólio profissional.
